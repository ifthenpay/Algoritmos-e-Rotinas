Para invocar a página de gerar referência multibanco em html:

http://www.oseuendereco.com/ifmb.html?ENTIDADE=[entidade]&SUBENTIDADE=[subentidade]&ID=[id]&VALOR=[valor]

Para poder testar e ver como fica use o nosso que já contem valores de teste:

http://www.ifthensoftware.com/multibanco/ifmb.html?ENTIDADE=11202&SUBENTIDADE=000&ID=1&VALOR=4.10

Nota: Os nomes das variáveis não devem ser alterados.


ficheiros necessários:

ifmb.js (modulo javascript para geração das referências)
ifmb.html (exemplo de html para mostrar o resultado)
logoMB.jpg (imagem logo multibanco)
